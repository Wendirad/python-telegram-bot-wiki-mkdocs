# python-telegram-bot-wiki-mkdocs
A python-telegram-bot wiki repository with mkdocs configuration to run locally.   

## Quick Start

### Requirments
* mkdocs==1.0.4
* mkdocs-windmill-dark==0.2.0

### Usage
- pip3 install -r requirments.txt
- run __mkdocs serve__
